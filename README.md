## Technology Stack

* **Framework**: Flutter
* **Language**: Dart
* **State Management**: Flutter Riverpod
* **Fonts**: Google Fonts (Poppins)

1. **Install Dependencies**:
   ```bash
   flutter pub get
   ```
2. **Run the App**:
   ```bash
   flutter run
   ```
